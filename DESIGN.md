# BizVault AI — Design System

## Visual Identity
- Futuristic AI OS aesthetic — dark, premium, glassmorphic
- Dark mode only
- Premium startup quality (Linear/Vercel level)

## Colors
- Background: #050508 base, #0D0D14 surface
- Glass cards: rgba(255,255,255,0.04) + backdrop-filter: blur(16px)
- Border: rgba(255,255,255,0.08)
- Primary Gradient: #7C3AED (purple) → #06B6D4 (cyan)
- Accent: #EC4899 (pink), #3B82F6 (blue)
- Success: #10B981, Warning: #F59E0B, Error: #EF4444
- Text: #F8FAFC primary, #94A3B8 secondary, #475569 muted

## Typography
- Font: Inter
- Bold headings, relaxed body, tabular nums for financials

## Components
- Glass cards: rgba bg + blur + subtle borders + 16px radius
- Neon glow buttons: purple→cyan gradient, white text
- Fixed left sidebar (240px), icon + label, active gradient bar
- Dashboard metric cards: large bold neon gradient numbers
- Recharts: dark bg, purple/cyan gradients, glass tooltips
- Dark tables with hover highlight
- Dark inputs with neon focus glow

## Layout
- Sidebar: 240px fixed desktop, hamburger mobile
- Content: flex-grow, 24px padding
- Responsive 12-column grid

## Animations
- Page transitions: fade + translateY 150ms
- Card hover: lift + glow
- Skeleton loading: purple shimmer
- Count-up on dashboard numbers

## Icons
- Lucide React throughout
